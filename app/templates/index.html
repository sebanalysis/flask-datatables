What's inside?

Open-Source sample crafted with Flask and Simple-Datatables JS Library that manages paginated data using different techniques.

  • DataTables - The information is loaded from database and injected in the HTML
  • DataTables API - Paginated information is lazy loaded from "/api/data" API Node
  • DataTables From File - The information is loaded a JSON file saved in "static/datatables" directory

How to use the code

The complete usage information is provided by README file shipped with the source code.
In case of any issues, feel free to contact .

# create a virtual environment
> $ virtualenv env
> $ source env/bin/activate
# Install Modules
> $ pip3 install -r requirements.txt
# Sert up Flask Environment
> $ export FLASK_APP=run.py
> $ export FLASK_ENV=development
# Load sample data (media/data.csv)
> $ flask load_data
# Start the APP
> $ flask run
